# Playwright Automation Project

This project contains Playwright automation tests for FPT Shop (fptshop.com.vn).

## Prerequisites

- Node.js (v14 or higher)
- npm

## Installation

Install the dependencies:

```bash
npm install
npx playwright install
```

## Running Tests

Run all tests:

```bash
npx playwright test
```

Run a specific test file:

```bash
npx playwright test tests/fptshop.spec.ts
```

Show the report:

```bash
npx playwright show-report
```
